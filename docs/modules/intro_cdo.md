# Introduction to CDO Tools

Climate Data Operators (CDO) is a collection of command-line operators to manipulate and analyze climate and NWP model data.

## Topics Covered
- What is CDO?
- Installation and setup
- Basic operators (`info`, `sinfo`, `show`)
- File operations (copying, merging, splitting)
- Mathematical operations (arithmetic, statistics)
- Grid operations (remapping, regridding)
- Working with NetCDF files
