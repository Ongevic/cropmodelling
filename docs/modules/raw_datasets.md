# Raw Datasets

This section provides access to raw datasets used in the course for exercises and projects.

## Available Datasets

### Climate Data
- **Historical Weather Data**: Daily temperature and precipitation records (1980-2020).
- **Climate Projections**: Downscaled climate model outputs for future scenarios (RCP 4.5, RCP 8.5).

### Crop Data
- **Field Trial Data**: Yield and biomass data from experimental sites.
- **Soil Data**: Soil properties and characteristics for simulation sites.
